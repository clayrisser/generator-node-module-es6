## Features

* ES6
* NPM best practices


## Installation

```sh
npm install -g yo generator-node-module-es6
```


## Dependencies

* [NodeJS](https://nodejs.org)
* [Yeoman](http://yeoman.io)


## Usage

```sh
yo node-module-es6
```
